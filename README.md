# CollegAid-Navigator

## Overview
CollegeAid Navigator is a student-built website designed to help families understand financial aid, FAFSA, and scholarships with clarity and confidence.

## Features
- FAFSA step-by-step guide
- Scholarship finder
- Budgeting tools for college costs

## Future Plans
- Location-based scholarship search
- Interactive calculators
- Expanded resources for first-generation families
